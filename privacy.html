Privacy Policy

SeriesGuide and its services are provided by Uwe Trottmann in Germany. See the Impressum for contact details.

The legal basis for processing personal data in regard to the GDPR is Article 6 lit. a - c and f.

Services collecting data

This section only applies to the SeriesGuide app. The separate X Pass app does not collect or share user data.

SeriesGuide uses its own and third-party services that collect and store user data.

Data collected by these services is transferred over a secure connection to, stored and processed in the United States, Germany and other countries where the services operate.

The Movie Database

The application programming interfaces (API) of The Movie Database (TMDB) are used to provide the content in the app. Therefore, IP addresses are sent to TMDB when accessing content.

You can find more details in their:

  • Privacy Policy
  • Terms of Use
  • API Terms of Use

Trakt

The application programming interfaces (API) of Trakt are used to provide content in the app and optionally enable connection of your Trakt profile. Therefore, IP addresses are sent to Trakt when accessing content or using Trakt features.

You can find more details in their:

  • Privacy Policy
  • Terms of Use

Firebase Crashlytics

This Google service:

  • Collects stack traces when the application crashes or an error occurs.
  • Collects relevant application state when the application crashes or an error occurs.
  • Collects point-in-time metadata about the device when the application crashes or an error occurs.
  • Generates and stores the Crashlytics installation UUID to measure the number of users impacted by a crash or error.

Crash and error traces and their associated identifiers are kept for 90 days.

This data is used to get insights into and measure the number of users impacted by issues with the app and to help fix them.

For more details have a look at:

  • Firebase data disclosure info about Crashlytics
  • Privacy and Security in Firebase
  • Firebase Crashlytics Terms of Service
  • Firebase Data Processing and Security Terms

Google Cloud Platform and Firebase Authentication

If you use SeriesGuide Cloud additional data is collected and stored by these Google services:

  • Collects your SeriesGuide data including shows, episodes, lists, movies, statistics.
  • Collects your email address to uniquely identify an account and its stored data.
  • Generates and stores request logs (including IP addresses and SeriesGuide version) to determine usage of and issues with the Cloud service.
  • Collects the Firebase user agent that includes device metadata, the app used to install the app (e.g. Play Store) and which Firebase SDKs are used. The Firebase user agent is used internally by Google to determine platform and version adoption in order to provide, maintain, and improve Firebase services. It is never linked to a user or device identifier.
  • Collects IP addresses to provide added security and prevent abuse during sign-up and authentication.
  • Collects user agent strings (that the app uses FirebaseUI, version of Authentication SDK, platform of the device) to provide, maintain, and improve Firebase services.
  • Collects the Firebase Android App ID of the app.
  • Generates and stores a unique Firebase Authentication identifier.

Request logs are kept for about 30 days. Your email address and SeriesGuide data is kept until you remove your SeriesGuide Cloud account from within SeriesGuide on all connected devices.

For more details have a look at:

  • Firebase data disclosure info about Authentication
  • Google Cloud Platform Terms of Service
  • Cloud Data Processing Addendum
  • Privacy and Security in Firebase

Image cache server

Images in the app are typically loaded from the SeriesGuide cache server. It may collect IP addresses and store them for a short period of time to provide added security, prevent abuse and ensure operation of the service.

Making a purchase or payment

When making a purchase (through Google Play, Amazon Appstore) or a payment (through PayPal) a record of the personal data you have supplied to the billing provider, like your name and address, may be processed and stored as long as needed to comply with legal obligations, like to determine taxes and comply with tax archival laws.

Contacting SeriesGuide

If you contact SeriesGuide to provide feedback, ask questions or ask for support a record of your communications, including your email address, is kept. This data is used when needed to provide support as requested by you and communicate with you. This data is not shared.

You can request your communication records to be deleted.

If contacted via email data is processed by Google Workspace by Google Cloud EMEA Limited, see the Cloud Data Processing Addendum.

Using this website

This website is hosted by GitHub Pages. If it is visited, the visitor's IP address is logged and stored for security purposes by GitHub.

This website uses Cloudflare for DNS and may use it for proxying requests. If it is visited, data processed by Cloudflare may include but is not limited to IP addresses, traffic routing data, system configuration information, and other information about traffic to and from this website.

Your Rights

You can request access to and rectification or erasure of your personal data, or the restriction of processing concerning you or to object to processing. You have a right to portability of your personal data. You can lodge a complaint with a supervisory authority regarding the processing of personal data.

Personal data means any information relating to you. Where you can be identified, directly or indirectly, in particular by reference to an identifier such as a name, an identification number, location data, an online identifier or to one or more factors specific to your physical, physiological, genetic, mental, economic, cultural or social identity.

To exercise these rights as relevant to SeriesGuide you can

  • Export your shows, lists and movies to files in JSON format using the backup tool in SeriesGuide settings.
  • Change the email address of your SeriesGuide Cloud account.
  • Delete your SeriesGuide Cloud account.
  • Send an email for any additional inquiries.
