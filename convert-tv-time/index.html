TV Time to SeriesGuide JSON Transformer

This is an experimental tool (source code). If there are issues, or you have a file format that you want supported, please leave feedback on the forum.

How to use this:

  • Export data using the TV Time Out by Refract browser extension. For shows, the converter also works with the JSON created by the Tv Time Liberator extension, but it's currently broken.
  • Upload a JSON file and select the type:
    • Shows: select your tvtime-series-2026-xx-xx.json (or shows.json) file and select "TV Time Out/Liberator shows JSON"
    • Lists (shows only): select your tvtime-lists-2026-xx-xx.json file and select "TV Time Out lists JSON"
  • Review the Input.
  • When converting shows, select the "Preferred show language" for content (titles, descriptions).
  • Select the "Transform" button.
  • Review the Output.
  • Select "Download JSON" at the bottom and save the file to your device.
  • Import the files as "Shows" or "Lists" in SeriesGuide.
