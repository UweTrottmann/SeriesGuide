TV Time to SeriesGuide JSON Transformer

This is an experimental tool (source code). If there are issues, or you have a file format that you want supported, please leave feedback on the forum.

Note: make sure you have SeriesGuide 2026.3.0 or newer installed. Check the version inside the app at the bottom of the "More" screen.

Note: movies that have no IMDB ID or where the IMDB ID is not set as external ID on TMDB can't be imported into SeriesGuide.

How to use this:

  • Take the JSON files you have previously exported using the TV Time Out by Refract browser extension. If needed, extract the ZIP archive to access the JSON files inside.
  • For each type, shows, movies and lists, create a transformed JSON file:
    • select the file type,
    • upload the required files from the export,
    • for shows specify the preferred content language,
    • select the "Transform" button,
    • review any skipped items and the preview of the transformed file,
    • select "Download JSON" at the bottom and save the file to your device.
  • Import the files as "Shows", "Lists" and "Movies" in SeriesGuide 2026.3.0 or newer (see note at the top).
