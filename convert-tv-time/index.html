TV Time to SeriesGuide JSON Transformer

This is an experimental tool (source code). If there are issues, please leave feedback on the forum.

How to use this:

  • Export data using the Tv Time Liberator Chrome extension (source code).
  • Select the shows.json file it created below.
  • Review the Input.
  • Select the "Transform" button.
  • Review the Output.
  • Select "Download JSON" at the bottom.
