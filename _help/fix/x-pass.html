--- layout: help title: "Get or install SeriesGuide X Pass (one-time purchase)" ---

ℹ️ X Pass is currently not available for new users. See how to unlock all features.

If you have purchased the standalone SeriesGuide X key app in the past, make sure to sign in with the account that bought it in Play Store and follow this link.

If you have purchased X Pass inside the app in the past, your purchase gets automatically restored. If that does not work, try the same troubleshooting steps as for subscriptions.

I decided to remove the one-time purchase option as SeriesGuide has recurring costs (Cloud, my living costs) that can't be sustained with a purchase valid for a life-time. I may consider to add a one-time purchase option back that does not include access to Cloud. Your feedback on this is appreciated.
