--- layout: help title: "Restore your data from an Auto Backup" ---

Auto Backup automatically exports your shows, lists and movies every week into JSON files.

To restore an Auto Backup (if available):

  1. In the bottom bar, select More.
  2. Select Auto Backup.
  3. Select the Restore Auto Backup button. If it is disabled, there is no auto backup available to restore from.

🚨 Not recommended when using SeriesGuide Cloud or Trakt. Syncing with Cloud or Trakt may overwrite any restored backup. With SeriesGuide Cloud, you can delete your account before restoring the backup, then re-create it afterwards to upload only the restored data.

To configure Auto Backup:

  1. In the bottom bar, select More.
  2. Select Auto Backup.
  3. Turn Auto Backup on or off.
  4. (Recommended) If turned on, enable the option to create copies, select files in an off-device location (e.g. Google Drive).

🚨 The Auto Backup files are saved to a storage directory that is removed when the app is uninstalled. Hence, it is recommended to turn on the option to create additional backups to another, at best off-device, location (e.g. Google Drive).

ℹ️️ Technical details: the Auto Backup files are stored in /Android/data/com.battlelancer.seriesguide/files/Backups. On Android 13 or higher, these can only be accessed using the system files app (shortcut available) or when connecting your device to a PC.
