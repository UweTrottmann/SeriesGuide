Export

To export your shows, lists and movies:

  1. In the bottom bar, tap More > Export and Import.
  2. Tap Export to file for either shows, lists or movies.

This will create so called JSON files. This uses a human and machine readable syntax. The format is documented.

To import your shows, lists and movies:

  1. In the bottom bar, tap More > Export and Import.
  2. Look for the Import section.
  3. Tap Select file for each type of data you want to restore and select a matching file.
  4. Tick the check box next to each type you want to restore.
  5. Tap Import data.

To import your data on another device, first transfer the created files to it. For example save the files to online storage, send yourself an email with the files attached or plug your device into a computer to access its storage.

Note You can import data from another source as long as it uses the format that SeriesGuide expects.
