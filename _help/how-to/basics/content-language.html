--- layout: help title: Change the language of a show ---

To change the language of a show already in your library:

  1. Tap on the show to view its overview screen.
  2. Swipe to the Details tab.
  3. Tap the globe symbol.
  4. Choose a language.

In More > Settings > Advanced you can set an Alternative Language. The alternative language will be used if no translation or poster in the language chosen for a particular show is available.

ℹ️ If details in your language are missing, help add them at the data source.

ℹ️ SeriesGuide adds a show in the language that is currently set as the search language (using the globe symbol in the top app bar). You can override the language by tapping on a show to view its details, then changing the language using the globe symbol before tapping Add show.
